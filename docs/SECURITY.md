# NexusVPN Security

## Overview

Security is a core component of NexusVPN.

---

## Encryption

- AES-256 for data encryption
- Secure key exchange mechanisms

---

## Protection Features

- Kill switch support
- DNS leak protection
- Secure tunnels using TUN interface

---

## Best Practices

- Always use strong passwords
- Keep software updated
- Monitor logs regularly

---

## Threat Model

Protects against:
- Eavesdropping
- Data tampering
- Network attacks
